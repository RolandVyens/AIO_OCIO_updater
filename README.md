# AIO-OCIO Updater

Blender addon to install and update the [AIO-OCIO](https://github.com/RolandVyens/AIO-OCIO) color configuration.

## Features

- One-click install/update from GitHub
- Progress bar showing download status
- Automatic backup of existing colormanagement folder
- Panel located below Color Management in Render Properties

## Installation

1. Install this addon in Blender (Edit > Preferences > Add-ons)
2. Enable the addon
3. Go to Render Properties > AIO-OCIO Updater panel
4. Click "Install AIO-OCIO"
5. Restart Blender to apply the new color configuration

## Requirements

- Blender 4.1+
- Internet connection (to download from GitHub)
